ngx_waf

Handy, High performance Nginx firewall module.

Quick Start →

Complete

Includes the basic functions of an application firewall, such as URL inspection, POST inspection, and CC defense.

Easy

Configuration files and rules files are simple to write.

Fast

The IP detection is a constant-time operation. Most of the remaining inspections use caching to improve performance.
